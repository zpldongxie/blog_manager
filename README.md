# blog_manager

个人博客管理后台

# 相关技术

该项目使用[Ant Design Pro](https://pro.ant.design)初始化。

## 提供的脚本

Ant Design Pro提供了一些有用的脚本，可帮助您快速启动和构建Web项目，代码样式检查和测试。

package.json中提供的脚本。

### 启动项目

```bash
npm start
```

### 构建项目

```bash
npm run build
```

### 检查代码样式

```bash
npm run lint
```

您还可以使用脚本自动修复某些lint错误：

```bash
npm run lint:fix
```

### 测试代码

```bash
npm test
```
